1. Engine솔루션 우클릭 - 속성
출력 디렉터리 : $(SolutionDir)Libraries\Lib\Engine\
중간 디렉터리 : $(SolutionDir)Intermediate\$(Configuration)\
C/C++ - 일반 - 추가 포함 디렉터리 : $(SolutionDir)Libraries\Include\;
C/C++ - 미리 컴파일된 헤더 : 사용, pch.h
라이브러리 관리자 - 일반  - 추가 라이브러리 디렉터리 : $(SolutionDir)Libraries\Lib\;
빌드 이벤트 - 빌드 전 이벤트 - 명령줄 (헤더파일과 inl파일 별도로 몰아두기(복사), 참조하기 위함)
xcopy /Y "$(SolutionDir)Engine\*.h" "$(SolutionDir)Libraries\Include\Engine"   (Engine폴더에 있는 .h파일을 모두Libraries\Include\Engine폴더에 복사한다 )
xcopy /Y "$(SolutionDir)Engine\*.inl" "$(SolutionDir)Libraries\Include\Engine"


2. Client솔루션 우클릭 - 속성
출력 디렉터리 : $(SolutionDir)Libraries\Lib\Binaries\

C/C++ - 일반 - 추가 포함 디렉터리 : $(SolutionDir)Libraries\Include\;      $(SolutionDir)Libraries\Include\Engine\(직접 제작한 기능도 사용하기 위함)
C/C++ - 미리 컴파일된 헤더 : 사용, pch.h
링커 - 일반  - 추가 라이브러리 디렉터리 : $(SolutionDir)Libraries\Lib\;
